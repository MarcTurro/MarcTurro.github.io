    Pour tracer un graphique quelconque, il faut en g√©n√©ral une liste pour les abscisses et une pour les ordonn√©es. Attention, ces deux listes doivent avoir le m√™me nombre d'√©l√©ments!

    La courbe repr√©sentative d'une fonction math√©matique

    Rien de plus simple! Il faut:

    • une fonction \(f\) d√©finie sur un intervalle \(I\)
    • une liste en abscisse
    • une liste en ordonn√©e

    Puis on utilisera la m√©thode .plot √† la place de .bar pour obtenir une courbe au lieu d'un histogramme...

    Par exemple voici le code python pour construire la courbe repr√©sentative des fonctions \(\cos\) et \(\sin\) sur l'intervalle \(I=[0, 2\pi]\).

    Fonctions trigonom√©triques

    ####### import biblioth√®ques ###############
import numpy as np
import math
import matplotlib.pyplot as plt
####### une figure et des axes ############
fig, ax = plt.subplots()
###### valeurs extr√™mes en X et Y #########
x_min = 0
x_max = 2*np.pi
y_min = -1 #valeur minimale du cos et sin
y_max = 1 #valeur maximale du cos et sin
##### fonction cos et sin #################
def lesinus(x):
    return np.sin(x)
def lecosinus(x):
    return np.cos(x)
##### g√©n√©ration de la liste des abscisses
X = np.arange(start = x_min, stop = x_max, step = 0.25)
#### limite horizontale du graphique ######

#### limite verticale du graphique ########
plt.ylim(y_min, y_max)
###### pour les grilles ###################
grid_x_ticks = np.arange(x_min, x_max, 0.5)
grid_y_ticks = np.arange(y_min, y_max, 0.25)
###### pour les graduations
ax.set_xticks(grid_x_ticks, minor= False)
ax.set_yticks(grid_y_ticks , minor= False)
#### pour l'affichage de la grille ########
ax.grid(which='both')
### construction des deux courbes
# X est la premi√®re liste et lesinus(X) est la deuxi√®me
plt.plot(X, lesinus(X), color ='red', label ='cos')
plt.plot(X, lecosinus(X), color = 'green', label ='sin')
##### pour la l√©gende #####################

plt.ylabel(r" $\sin(x)$ et $\cos(x)$")#syntaxe latex

plt.title(r"Fonctions trigonom√©triques ")
plt.show()
